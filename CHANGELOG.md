# Changelog

## 1.1.0 - 2026-07-13

### 新增

- 增加浏览器端视频抽帧演示，支持上传视频、预览抽帧结果并导出帧图片。

### 变更

- 将演示界面中的 Element Plus 控件替换为原生控件实现，减少 UI 组件库依赖。
- 调整忽略规则，排除本地文档目录。

## 1.0.0 - 2026-07-10

### 初始发布

- 初始化工业视觉数据集工具箱项目。
- 提供图像预处理体检、图片元数据读取、数据清洗、清晰度分析、样本增强与模型推理演示。
- 添加 YOLO/Pointer 示例数据集、COCO 转 YOLO 辅助脚本与感知哈希原理文档。
- 配置 Vue 3 + Vite 基础工程、构建脚本与项目说明文档。
